# Haresh Gir — Portfolio

Personal portfolio website built with Next.js and deployed on Vercel.

## Tech Stack

- Next.js
- React.js
- React-Bootstrap
- CSS3

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
